#  Project Overview  

This project demonstrates a **menu-driven File Handling System** developed in C programming.  
It performs fundamental file operations such as **creation, reading, writing, and appending data** using standard file handling functions.

The system showcases how data can be persistently stored and managed using files in C.


#  Objectives  

- To understand file handling concepts in C  
- To implement file operations using standard libraries  
- To work with file pointers and file modes  
- To design a structured menu-driven application  



#  Functionalities  

The program supports the following operations:

Create a new file  
Write data into a file  
 Append data to an existing file  
Read and display file content  
Exit the program  



#  Key Concepts Used  

- File pointer (`FILE *`)  
- File opening modes (`r`, `w`, `a`)  
- `fopen()` – Open/Create file  
- `fclose()` – Close file  
- `fgets()` – Read input string  
- `fputs()` – Write string to file  
- `fgetc()` – Read character from file
